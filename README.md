# ESDM Virtual Lab (PICT)

A cross-platform educational platform developed for the **Entrepreneurial Software Development and Management (ESDM)** course at **Pune Institute of Computer Technology (PICT)**.

The platform enables students and faculty to efficiently manage academic resources, assignments, quizzes, and performance analytics through both web and mobile interfaces.

---

## Features

### User Authentication & Role Management

* Secure login and registration system.
* Role-based access control for:

  * Students
  * Faculty
  * Administrators

### Content Management

* Faculty can upload and manage notes and study materials.
* Students can access and download learning resources.

### Assignment Module

* Assignment creation with deadlines and descriptions.
* Online assignment submission.
* Faculty evaluation and feedback system.

### Quiz Module

* Multiple-choice quiz creation.
* Automated quiz evaluation.
* Instant score generation.
* Performance tracking.

### Analytics Dashboard

* Student performance insights.
* Assignment completion statistics.
* Quiz score analytics.
* Academic progress monitoring.

### Cross-Platform Support

* Web Application
* Android Application (React Native)

## Project Objective

The objective of this project is to provide a centralized digital learning environment for the ESDM course at PICT, improving resource sharing, assignment management, assessment processes, and academic performance tracking.
